Z80 Assembly Language


Z80 Assembly Language is the computing language I am the most experienced in, and is the only one I would really say
I am "fluent" in. It's a very simple language, but that simplicity makes it quite difficult to get anything useful done.
Here's an example program I made to convert the number stored in the HL register to an ASCII string:

9400HLDIVC: LD B,16
9405 XOR
9410D0: ADD HL,HL
9415 RLA
9420 CP C
9425 JR C,D1
9430 INC L
9435 SUB C
9440D1: DJNZ D0
9445 RET
9449; CONV HL TO ASCII
9450ASCII: LD DE,NUM+3 ; NUM is the address where we want to store the ASCII string

; The Z80 stores the data with the least significant byte first,
; the opposite of how our decimal number system works. Because of
; this, you have to put the larger part of the number at the end of
; the address space alloted for this function to return to. Hence, NUM+3
; being used instead of just NUM.

9455 LD C,10
9460AS0: CALL HLDIVC
9470 LD (DE),A
9475 DEC DE
9476 XOR A
9478 LD (DE),A
9480 CP L
9485 JP NZ,AS0
9490 RET

And there you go!
